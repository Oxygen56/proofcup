# ZK Artifacts

`matchpass/` contains proof outputs generated from the Noir circuit:

- `proof.json`: UltraHonk proof generated by Barretenberg.
- `public_inputs.json`: public inputs for `team_id`, `tournament_id`, `commitment`, and `nullifier`.
- `vk.json`: verification key.

Regenerate and verify:

```bash
cd proofcup
pnpm nargo:compile
pnpm nargo:execute
pnpm proof:prove
pnpm proof:verify
```
